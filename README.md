# AI Network Analyzer Dashboard

## 🚀 Overview
An AI-powered network analysis tool that processes PCAP files and visualizes network traffic insights.

## 🧠 Features
- Packet capture analysis (PCAP)
- AI-based traffic classification
- Network anomaly detection
- Dashboard visualization

## 🛠 Tech Stack
- Python
- Machine Learning
- Pandas / Scikit-learn
- Streamlit / Flask (if used)
- Wireshark PCAP analysis

## 📂 Project Structure
- dashboard.py → main dashboard
- models/ → trained ML models
- capture_*.pcap → sample network data

## 📊 Use Case
Cybersecurity + Network monitoring + Traffic analysis
