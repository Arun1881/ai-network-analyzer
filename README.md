# AI Network Problem Analyst

Real-time network monitoring dashboard built with Python, Dash, Plotly, Pandas, and Scapy.

## Features

- Live Packet Capture
- Source/Destination IP Analysis
- Protocol Distribution
- Geo-IP Visualization
- Network Traffic Prediction
- CSV Export
- Alerts & Monitoring

## Technologies

- Python
- Dash
- Plotly
- Pandas
- Scapy

## Run

pip install -r requirements.txt

python dashboard.py
